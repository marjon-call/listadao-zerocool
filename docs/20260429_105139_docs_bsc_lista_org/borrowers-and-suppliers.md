### [hashtag](https://docs.bsc.lista.org/zh-cn/jian-jie/lista-lending/borrowers-and-suppliers\#gong-ying-shang-chu-jie-ren)    供应商（出借人）

任何人都可以成为供应商——个人、协议、DAOs 或甚至对冲基金。他们可以通过手动调整在不同市场中的敞口来积极管理自己的头寸，或者通过将资金存入与其风险配置相符的金库来采取更被动的方式。然后，金库将为供应商在各个市场分配资金。

### [hashtag](https://docs.bsc.lista.org/zh-cn/jian-jie/lista-lending/borrowers-and-suppliers\#jie-kuan-ren)    借款人

借款人可以根据自己的需求在 Lista Lending 上选择一系列市场，如首选的抵押品类型、贷款资产和有利的借款利率。
