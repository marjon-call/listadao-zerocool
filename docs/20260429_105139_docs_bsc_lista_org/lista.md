LISTA 是 Lista DAO 的治理代币。它允许用户参与协议治理，例如在 Lista DAO 的 [Snapshot 平台arrow-up-right](https://snapshot.org/#/listavote.eth)上提出提案和对其进行投票。

与 lisUSD 类似，LISTA 代币本身也是一种交易媒介，用户可以根据自己的判断在 DEX 上进行交换或放入农场池，例如为了最大化资本效率、获得借款和/或复合他们的收益。

根据 [LIP-021arrow-up-right](https://snapshot.org/#/s:listavote.eth/proposal/0xdb92580e04ca8d688404a689a99579294715bd9ab858cc9b7ba637599720a6ff)，LISTA 的总供应量已从 10 亿减少到 8 亿。在 CoinMarketCap 上查看我们的信息 [这里arrow-up-right](https://coinmarketcap.com/currencies/lista-dao/)。

经 [LIP-024arrow-up-right](https://snapshot.org/#/s:listavote.eth/proposal/0x1a15347f6b452049212bdf51ff1a46c0a7edf7ca8efe1004b32c15c2965f0f3b) 批准，实施了新的代币经济模型。此升级包括：

- 终止 veLISTA 质押模型。所有质押的 LISTA 可以在[此页面arrow-up-right](https://lista.org/lock)无罚金解锁。

- 之前分配给 veLISTA 质押者的协议收入现在将资助 $LISTA 回购，继续我们将 LISTA 变为通缩代币的努力。

- 之前分配给 veLISTA 质押者的排放将继续支持协议，以资助用户奖励、增长计划和生态系统活动。

- 从现在开始，所有 LISTA 持有者都可以对 LIP 提案进行投票。作为这一过渡的一部分，LP 池投票系统已被取消。


未来，将为 LISTA 持有者引入更多好处。列表上的第一个是延迟清算。请继续关注更多详情。
