通过将BNB委托给ListaDAO的验证者，可以获得质押奖励。验证者在将奖励分享给协议作为可认领代币之前，会先从中提取他们的部分。所赚取的奖励分为两部分：

- **95% 归slisBNB持有者**，以slisBNB价值增值的形式：slisBNB持有者的BNB池份额持续增加，因为slisBNB/BNB汇率的提高。

- **5% 归协议所有**。
